ⓘ What do Class A, B and C+ mean?

Class A: Highest-quality buildings. Usually newer, best locations, modern systems, strong amenities, high rents, low risk.

Class B: Good/average-quality buildings. Usually older than Class A, decent location and condition, average rents, moderate risk.

Class C+: Slightly better than Class C, but not quite Class B. It may have some improvements, better location, or fewer condition problems than a typical Class C property.
